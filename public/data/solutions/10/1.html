Gibt es in TypeScript statische Klassen? Erläutern Sie den Unterschied von TypeScript zu anderen objekt-orientierten, statisch typisierten Programmiersprachen wie Java und C#.
Nein, in Typescript gibt es keine statischen Klassen, man kann jedoch statische Methoden in Klassen definieren. In Typescript sind Typangaben optional, da in TS die Typen abgeleitet werden können. In Java muss immer eine Typangabe stehen. Typen von Java sind statisch und strongly typed. TS sind dynamisch und strongly typed (optional).
