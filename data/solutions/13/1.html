In der Vorlesung wurde ein ArrayWrapper Exploit gezeigt. Analysieren Sie die Ursache dieses Exploits. Entwickeln Sie ein Sicherheitskonzept und eine ArrayWrapper-Lösung, die gegen diesen Exploit gewappnet ist.
Ursachenanalyse:
Durch das Setzen im Array von ungeprüften Daten mit ungeprüften Typen, können Angreifer eine getarnte "push"-Funktion in das Array einschleusen, die diese durch die Funktion append() aufgerufen wird.

Sicherheitskonzept:
Man kann den Exploit vermeiden, indem man im Vorhinein prüft, von welchem Typ die Daten sind. Falls diese eine Funktion ist, wird das setzen in das Array automatisch abgelehnt.
