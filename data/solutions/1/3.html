Was ist HTML?
Der Begriff "HTML" steht für "Hypertext Markup Language". HTML ist die Standard Markdown Sprache für Webseiten.

Wie kann man eine geschachtelte geordnete Liste der Schachtelungstiefe 3 erzeugen? <ol> <li> <ol> <li>Coffee</li> <li> <ol> <li>Coffee</li> <li>Tea</li> <li>Milk</li> </ol> </li> <li>Milk</li> </ol> </li> <li>Tea</li> <li>Milk</li> </ol> Wie ist eine HTML-Tabelle aufgebaut?
Eine HTML Tabelle besteht aus Table Rows (tr), Table Cells (td) und Table Headers (th). Die erste Zeile einer HTML Tabelle besteht aus Table Headers Elementen, in den nachfolgenden Zeilen kommen die Table Cells Elemente.

Welche Konventionen sollte man bei Pfaden und Dateinamen beachten?
Kleinbuchstaben, keine Leerzeichen, nur Buchstaben und Zahlen, sowie nur die Sonderzeichen "-" und "_"

Wie baut man in HTML ein Menü?
Ein Menu baut man aus mehreren a (Link)-Elementen <ul> <li><a href="/">Home</a></li> <li><a href="/News">News</a></li> <li><a href="/Contact">Contact</a></li> <li><a href="/About">About</a></li> </ul> Welche Attribute sollte man bei Bildern wie verwenden?
src = Die Bild URL
alt = Ein Alternativer Text für das Bild
height = Die Höhe des Bildes
width = Die Breite des Bildes
